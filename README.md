# Random Quote Generator (Генератор случайных цитат)

**Автор:** Рямова Камилла

## 📖 Описание
GUI-приложение на Python (Tkinter) для генерации случайных цитат. Программа позволяет:
- Генерировать случайные цитаты из предустановленного набора.
- Добавлять собственные цитаты (с указанием автора и темы).
- Фильтровать историю цитат по автору и теме.
- Сохранять историю генераций в файл `quotes_history.json` и загружать её при следующем запуске.

## 🛠 Требования
- Python 3.6 или выше.
- Стандартная библиотека Tkinter (обычно входит в состав Python).
